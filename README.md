# Fuel Consumption Calculator

A simple C++ console program that calculates the fuel consumption for multiple trips.

## Features

- Input trips count, distance, and fuel used for each trip.
- Calculates total distance, total fuel used, and average consumption (L/100 km).
- Checks if fuel consumption is above normal.

## Requirements

- C++20 or newer
- Standard C++ library

## How to Compile and Run

### Using g++
```bash
g++ -std=c++20 src/main.cpp -o fuel_consumption
./fuel_consumption
